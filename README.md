# MapTableMaker

## About

Easily make and download high resolution SVG maps from your CSV dataset

## Requirements

A web server running PHP

## Install

1. Download latest to a web accessible directory
1. Browser to URL
1. Enter CSV values and submit form
1. Download map
1. Enjoy!
